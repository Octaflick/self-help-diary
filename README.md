# Self-Help Diary

## Purpose

The Self-Help Diary is designed to provide users with a digital platform to maintain a personal diary. It aims to help users structure their thoughts and reflections, enhanced by integrated self-help quotes for inspiration and guidance.

## APIs

The project plans to integrate multiple APIs in the future to enhance its functionalities. Currently subject to change upon further testing

## Features

- **Multi-page entries**: Users can create detailed diary entries spanning multiple pages.
- **Search functionality**: Easily search through past entries to find specific thoughts or topics.
- **Customizable themes**: Personalize the diary's appearance with light and dark themes.
- **Range of self-help quotes**: Various range of self-help thoughts to guide the user through their writing process.

## Technologies

- HTML
- CSS
- JavaScript
- React (Basic implementation)
